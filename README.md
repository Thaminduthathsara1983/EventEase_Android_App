# EventEase - Android Event Planning Application

EventEase is an Android application developed using Java in Android Studio. The application helps users manage events, view weather information, and receive activity suggestions through API integration.

## Features

* User Authentication using Firebase Authentication
* Event Management using Firebase Realtime Database
* Weather Information using a REST API
* Activity Suggestions using a public API
* Simple and user-friendly Android UI
* Multiple Activity Navigation

## Technologies Used

* Java
* Android Studio
* Firebase Authentication
* Firebase Realtime Database
* Volley Library
* REST APIs
* XML Layout Design
